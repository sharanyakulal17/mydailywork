# Project Management Tool
A full-stack project management web application built using **React**, **Node.js**, **Express**, and **MongoDB**.

## Features
- Create a new project
- Add tasks with title and deadline
- Track task status (**To Do**, **In Progress**, **Done**)
- Step-by-step project and task management
- Frontend and backend separated
- MongoDB database integration

## Tech Stack
### Frontend
- React
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Project Structure
```text
project-management-tool/
│
├── frontend/      # React frontend
├── backend/       # Node.js + Express backend
└── README.md
```

## How to Run the Project

### Backend
```bash
cd backend
npm install
node server.js
```
Runs on: http://localhost:5000

### Frontend
```bash
cd frontend
npm install
npm start
```
Runs on: http://localhost:3000
