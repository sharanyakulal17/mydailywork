# Online Quiz Maker

This is a simple Online Quiz Maker project built using HTML, CSS, and JavaScript.

## Features
- Create a quiz (maximum 5 questions)
- Multiple-choice questions
- Take quiz and select answers
- View final score after submission
- Simple and clean UI
- Mobile responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open the project folder
2. Open `index.html` in a browser
3. Create or take a quiz
